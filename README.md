# 📄 AI-Powered Grant Proposal Assistant

An AI-driven tool that helps researchers, students, and nonprofits **draft, evaluate, and refine grant proposals** using a multi-agent system powered by local LLMs.

---

## 🚀 Features

- 🧠 **Multi-Agent Architecture**
  - Outline Designer Agent
  - Budget Estimator Agent
  - Reviewer Simulation Agent  

- 📝 **Automated Proposal Generation**
  - Structured proposal outline
  - Budget estimation
  - Reviewer-style feedback

- 🔁 **Version Tracking (Memory System)**
  - Stores proposal history
  - Tracks revisions and rationale

- ⚡ **Local AI Processing**
  - Runs fully locally using LLaMA 2 via Ollama

---

## 🧩 Tech Stack

- Backend: FastAPI  
- Frontend: Streamlit  
- LLM Runtime: Ollama (LLaMA 2)  
- Language: Python  
- Storage: JSON (lightweight memory system)
